# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project is a Rule-Based AI Chatbot developed in Python as part of the DecodeLabs Artificial Intelligence Internship Program.

The chatbot interacts with users through predefined rules and responses. It demonstrates the basic concepts of Artificial Intelligence, user interaction, conditional logic, loops, functions, and pattern matching.

Unlike machine learning chatbots, this chatbot generates responses based on predefined conditions and user inputs.

---

## 🎯 Objectives

- Understand the fundamentals of conversational AI.
- Implement a rule-based chatbot using Python.
- Practice conditional statements and loops.
- Create an interactive command-line application.
- Simulate human-like conversation using predefined responses.
