# 🌸 Data Classification Using AI

## 📌 Project Overview

This project demonstrates the fundamentals of Machine Learning and Artificial Intelligence using a classification model. The system uses the Iris dataset to train a machine learning algorithm and predict flower species based on their features.

The project was developed as part of the DecodeLabs Artificial Intelligence Internship Program and focuses on understanding the complete machine learning workflow from data loading to model evaluation.

---

## 🎯 Objectives

- Understand the basics of supervised machine learning.
- Load and analyze a dataset.
- Split data into training and testing sets.
- Train a classification model.
- Make predictions using the trained model.
- Evaluate model performance.

## 📊 Dataset Used

### Iris Dataset

The Iris dataset is a popular machine learning dataset containing information about three species of Iris flowers:

- Setosa
- Versicolor
- Virginica

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🤖 Machine Learning Algorithm

### Decision Tree Classifier

A Decision Tree Classifier is used to classify flower species based on their measurements.

Benefits:

- Easy to understand
- Beginner friendly
- High classification accuracy
- Fast training process
